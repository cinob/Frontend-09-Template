学习笔记

```javascript
// 快速创建一个指定长度的数组并填充指定值
Array(100).fill(0)


// 解构赋值前(重新赋值, 非初始化), 前一行必须加 `;` 否则赋值无效
let [x, y] = [1, 2];
[x, y] = [3, 4]

// 在某些对数组顺序没有要求的场景, 可以替换Array.splice函数, 节省写操作的开销
const arr = [1, 3, 4, 6, 7, 8, 4, 2, 2, 2, 8]
// 移除索引为2的值;
// 只进行了一次写操作, 一次pop()操作
arr[2] = arr[arr.length - 1]
arr.pop()
```
